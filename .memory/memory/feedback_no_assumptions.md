name: feedback_no_assumptions
description: Never assume what the user wants to do next — always wait for direction
type: feedback
---

Never assume or suggest what the user wants to do next.

**Why:** User explicitly stated nobody (human or machine) is ever correct in their assumptions about next steps, and finds it unwelcome.

**How to apply:** After completing a task or reaching a decision point, stop and wait for the user to direct the next step. Do not propose "shall we start Phase 1?" or similar. Just confirm completion and wait.
